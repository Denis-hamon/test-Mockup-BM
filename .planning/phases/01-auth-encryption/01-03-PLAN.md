    **Keypair** (`packages/crypto/src/keypair.ts`):
    Uses `libsodium-wrappers-sumo`. MUST await `sodium.ready` before any operation.
    - `generateKeypair()`: calls `sodium.crypto_box_keypair()`, returns `{ publicKey, secretKey }` as `KeyPair`
    - `publicKeyToBase64(key: Uint8Array)`: returns `sodium.to_base64(key, sodium.base64_variants.ORIGINAL)`
    - `base64ToKey(b64: string)`: returns `sodium.from_base64(b64, sodium.base64_variants.ORIGINAL)`

    **Encrypt** (`packages/crypto/src/encrypt.ts`):
    Uses XChaCha20-Poly1305 (AEAD, authenticated encryption).
    - `encrypt(plaintext: Uint8Array, key: Uint8Array)`: generates random nonce via `sodium.randombytes_buf(sodium.crypto_secretbox_NONCEBYTES)`, encrypts with `sodium.crypto_secretbox_easy(plaintext, nonce, key)`, returns `{ ciphertext, nonce }`
    - `decrypt(ciphertext: Uint8Array, nonce: Uint8Array, key: Uint8Array)`: calls `sodium.crypto_secretbox_open_easy(ciphertext, nonce, key)`, returns plaintext Uint8Array. Throws on failure (wrong key/nonce/tampered data).

    **KDF** (`packages/crypto/src/kdf.ts`):
    Argon2id key derivation via `sodium.crypto_pwhash`. MUST use `libsodium-wrappers-sumo` (NOT standard build -- standard lacks crypto_pwhash, per research Pitfall 1).
    - `generateSalt()`: returns `sodium.randombytes_buf(sodium.crypto_pwhash_SALTBYTES)` (16 bytes)
    - `deriveKey(password: Uint8Array | string, salt: Uint8Array)`: derives 32-byte key using `sodium.crypto_pwhash(32, password, salt, sodium.crypto_pwhash_OPSLIMIT_MODERATE, sodium.crypto_pwhash_MEMLIMIT_MODERATE, sodium.crypto_pwhash_ALG_ARGON2ID13)`. If password is string, convert with `sodium.from_string(password)` first.
    - Export constants: `KDF_OPSLIMIT = sodium.crypto_pwhash_OPSLIMIT_MODERATE`, `KDF_MEMLIMIT = sodium.crypto_pwhash_MEMLIMIT_MODERATE`, `KDF_ALG = sodium.crypto_pwhash_ALG_ARGON2ID13`


    Functions:
    - `generateRecoveryMnemonic()`: `await sodium.ready; return bip39.generateMnemonic(wordlist, 128);` — 128 bits entropy = 12 words (D-01)
    - `encryptPrivateKey(secretKey: Uint8Array, mnemonic: string)`: generates salt via `generateSalt()`, derives key from mnemonic via `deriveKey(mnemonic, salt)`, encrypts secretKey via `encrypt(secretKey, derivedKey)`, returns `RecoveryBundle` with base64-encoded ciphertext, nonce, salt, and KDF params
    - `decryptPrivateKey(bundle: RecoveryBundle, mnemonic: string)`: decodes base64 salt/nonce/ciphertext, derives key from mnemonic with same salt, decrypts, returns secretKey Uint8Array. Throws if mnemonic is wrong (D-02: no recovery)
    - `selectRandomWordIndices(count = 3)`: Pick `count` unique random indices from 0-11, return sorted array. Use `sodium.randombytes_uniform(12)` for cryptographic randomness (not Math.random).
    - `verifyPassphraseWords(mnemonic: string, attempts: Record<number, string>)`: split mnemonic into words, check each attempt[index] matches words[index] (case-insensitive, trimmed). Return boolean. (D-04)
