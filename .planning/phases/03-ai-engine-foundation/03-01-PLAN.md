       - createProvider(type): returns Vercel AI SDK LanguageModel

       - `providers/openai.ts`: createOpenAIModel(modelId)
       - `providers/mistral.ts`: createMistralModel(modelId)

    7. Create streaming helper (`packages/ai/src/stream.ts`):
       - streamAIResponse(options): wraps Vercel AI SDK streamText
       - Automatically appends legal disclaimer to system prompt
       - Accepts messages, model config, system prompt

    8. Create barrel export (`packages/ai/src/index.ts`)

    9. Add @legalconnect/ai as dependency in apps/web/package.json

    10. Create `apps/web/src/lib/ai.ts`:
        - Re-export convenience functions from @legalconnect/ai
        - App-level AI configuration
