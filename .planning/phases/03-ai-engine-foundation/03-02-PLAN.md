**src/guardrails/rewrite.ts** — Per D-01, LLM-based rewrite:

Export `rewriteWithoutAdvice(text: string): Promise<string>` that:
1. Calls `generateText` from `ai` with a fast model (`openai:gpt-4.1-mini`) using a dedicated rewrite system prompt
2. The rewrite prompt instructs: remove all legal advice, keep factual/empathetic content, add redirection to lawyer ("Votre avocat sera la personne la plus qualifiee pour vous repondre sur ce point.")
3. Mark the output as `_rewritten: true` in a WeakSet or similar to prevent re-checking (avoid infinite loop per Pitfall 3)
4. If the rewrite call fails, return a hardcoded safe fallback: "Je comprends votre question. Ce sujet necessite l'expertise de votre avocat, qui pourra vous donner un avis adapte a votre situation. Souhaitez-vous que je vous aide a preparer votre dossier pour votre avocat?"

For testing: export a `_setRewriteModelForTest(model)` function so tests can inject a mock model.

**src/guardrails/upl-middleware.ts** — Per D-01, the core middleware:

Export `uplGuardrailMiddleware` as `LanguageModelV3Middleware`:
- `wrapGenerate`: call `doGenerate()`, if `result.text` and `containsLegalAdvice(result.text)`, call `rewriteWithoutAdvice(result.text)` and replace `result.text`. Then apply `shouldAddDisclaimer`/`addDisclaimer`.
- `wrapStream`: Use full-buffer approach for v1 (per Research open question 1 recommendation). Collect the full stream, check with `containsLegalAdvice`, rewrite if needed, then emit the (possibly rewritten) text. Apply disclaimer. NOTE: this buffers but the model still generates in parallel so it is faster than synchronous generateText.

Import from `ai`: `LanguageModelV3Middleware`.
