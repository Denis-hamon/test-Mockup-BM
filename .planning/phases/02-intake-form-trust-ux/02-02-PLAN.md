export const stepProblemTypeSchema = z.object({

Create `apps/web/src/hooks/use-intake-form.ts` (per D-03: auto-save, no account required to start):

- "use client" directive
- Import `useForm` from `react-hook-form`, `zodResolver` from `@hookform/resolvers/zod`
- Import `intakeSchema`, `STEP_SCHEMAS`, `IntakeFormData` from `@legalconnect/shared`
- Import `useEffect`, `useCallback`, `useState` from `react`

Constants:
- `STORAGE_KEY = "legalconnect_intake_draft"` (per UI-SPEC Auto-Save Behavior)
- `TOTAL_STEPS = 4`

`loadDraft()` function:
- If `typeof window === "undefined"`, return `{}`
- Try to parse `localStorage.getItem(STORAGE_KEY)`, return parsed or `{}`
- Catch errors, return `{}`

`useIntakeForm()` hook returns `{ form, currentStep, setCurrentStep, validateStep, nextStep, prevStep, hasDraft, clearDraft }`:
- `form = useForm<IntakeFormData>({ resolver: zodResolver(intakeSchema), defaultValues: loadDraft(), mode: "onTouched" })`
- `currentStep` state (number, initial: load from localStorage `legalconnect_intake_step` or 0)
- `hasDraft` state (boolean, true if localStorage had saved data on mount)
- `storageFull` state (boolean, for QuotaExceededError handling)

Auto-save effect:
```typescript
useEffect(() => {
  const subscription = form.watch((values) => {
    try {
      localStorage.setItem(STORAGE_KEY, JSON.stringify(values));
    } catch (e) {
      if (e instanceof DOMException && e.name === "QuotaExceededError") {
        setStorageFull(true);
      }
    }
  });
  return () => subscription.unsubscribe();
}, [form]);
```

Also save currentStep to localStorage:
```typescript
useEffect(() => {
  localStorage.setItem("legalconnect_intake_step", String(currentStep));
}, [currentStep]);
```

`validateStep(stepIndex: number)`:
- Get field names from `Object.keys(STEP_SCHEMAS[stepIndex].shape)`
- Call `form.trigger(fields)` and return the boolean result

`nextStep()`:
- Call `validateStep(currentStep)`. If true and currentStep < TOTAL_STEPS - 1, increment currentStep.
- Return validation result.

`prevStep()`:
- If currentStep > 0, decrement. No validation on back navigation (per UI-SPEC Stepper Behavior).

`clearDraft()`:
- `localStorage.removeItem(STORAGE_KEY)`
- `localStorage.removeItem("legalconnect_intake_step")`
- `form.reset({})`
- `setCurrentStep(0)`
- `setHasDraft(false)`
