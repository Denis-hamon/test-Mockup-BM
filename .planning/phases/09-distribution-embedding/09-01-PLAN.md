    - Immediately capture `document.currentScript` at top level (before any async). Fallback: `document.querySelector('script[data-slug]')` if null (handles async/defer).
    - Read data-* attributes: data-slug (required), data-color (optional, default "#1a365d"), data-position (optional, default "bottom-right").

    - Create host div, append to document.body, attachShadow({ mode: "open" }).
    - Import widget.css with `?inline` query, inject as `<style>` element into shadow root.
    - Compute accent foreground via luminance.ts (white if accent luminance < 0.5, else dark).
    - Set CSS custom properties on shadow host: --lc-accent, --lc-accent-fg, --lc-surface, --lc-fg, --lc-muted, --lc-border.
    - createRoot on mount div inside shadow root, render Widget component with config props.
    - Button appears with 300ms delay fade-in after script loads.

    - submitWidgetIntake(slug: string, data: Record<string, unknown>): POST /api/widget/submit, body: { slug, responses }.
    - All fetch calls include appropriate error handling.

    **5. Create lib/luminance.ts:**

    - .lc-backdrop: rgba(0,0,0,0.5) mobile only.
    - Form elements: .lc-input, .lc-label, .lc-select, .lc-textarea, .lc-field-error.
    - .lc-btn-primary (accent), .lc-btn-secondary (outlined).

    - `/api/widget/route.ts` (GET): Read widget.js from `../widget/dist/widget.js` (resolve path). Return with Content-Type application/javascript, CORS headers (Access-Control-Allow-Origin: *), cache control (immutable if ?v= param, else max-age=3600).
    - `/api/widget/template/[slug]/route.ts` (GET + OPTIONS): Internally query DB for template by slug (reuse getTemplateBySlug logic but as direct DB query, not server action). Return template JSON with CORS headers. 404 if not found.
    - `/api/widget/submit/route.ts` (POST + OPTIONS): Accept { slug, responses }, create intake submission in DB (reuse existing intake submission logic). Return 201 with { success: true, id }. CORS headers on all responses.
    - Create shared corsHeaders() helper used by all widget API routes.
