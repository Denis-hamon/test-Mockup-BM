# Phase 8 — UI Design Contract

> Visual and interaction contract for intake templates and lawyer customization. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn/ui CLI v4 |
| Preset | base-nova (neutral base) |
| Component library | Radix UI (via shadcn) |
| Icon library | Lucide |
| Font | Geist (variable --font-sans, loaded via next/font/google) |

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding, drag handle spacing |
| sm | 8px | Compact element gaps, question list item padding |
| md | 16px | Default element spacing, card padding inline, form field gaps |
| lg | 24px | Section padding, card padding block, editor panel padding |
| xl | 32px | Layout gaps between editor and preview panels |
| 2xl | 48px | Page section breaks |
| 3xl | 64px | Not used in this phase |

Exceptions: 44px minimum touch target for drag handles on mobile and color picker swatches.

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px | 400 (regular) | 1.5 | Question labels in editor, preview form fields, template descriptions |
| Label | 14px | 600 (semibold) | 1.4 | Section headers in editor, field type labels, tab labels, "Propulse par LegalConnect" |
| Heading | 20px | 600 (semibold) | 1.2 | Page titles ("Personnaliser le formulaire"), template names, preview header |
| Display | 28px | 600 (semibold) | 1.2 | Not used in this phase (reserved for landing) |

Note: Only 2 weights used -- 400 (regular) for body/content, 600 (semibold) for labels/headings/emphasis. Consistent with Phases 6-7.

### Lawyer Accent Override (Client-Facing Intake)

On `/intake/[slug]` pages, the lawyer's custom accent color replaces `--primary` via CSS custom properties injected at page level. Fallback: default `--primary` if no accent configured.

| Property | CSS Variable | Default |
|----------|-------------|---------|
| Accent color | --lawyer-accent | oklch(0.205 0 0) |
| Accent foreground | --lawyer-accent-foreground | oklch(0.985 0 0) |

Applied to: stepper progress indicator, primary CTA button ("Soumettre ma demande"), active step highlight on client-facing intake form only.

Accent reserved for: primary CTA buttons, selected template card border, active tab indicator, stepper progress fill. Never for decorative elements.

## Interaction Contract

### Template Selection

| Interaction | Behavior |
|-------------|----------|
| Template card click | Select template, highlight with primary border. Other cards deselect. |
| "Utiliser ce template" button | Load template into editor, navigate to /settings/cabinet/template/edit |
| Card hover | Subtle elevation (shadow-sm to shadow-md transition, 150ms) |

### Template Editor

| Interaction | Behavior |
|-------------|----------|
| Drag question | @dnd-kit/sortable vertical list. Drag handle (GripVertical icon) activates drag. Drop indicator: 2px primary border between cards. Preview updates in real-time. |
| Click question card | Expand inline to show QuestionEditForm. Collapse other expanded cards. |
| Add question | Ghost button at bottom. Adds new question card at end of list, auto-expands for editing. |
| Delete question | Trash icon on question card. Alert-dialog confirmation. Question removed, preview updates. |
| Edit question text | Inline input, debounced 300ms update to preview. |
| Change field type | Select dropdown. Preview updates immediately with correct field rendering. |
| Toggle required | Switch. Adds asterisk (*) in preview next to question label. |
| Toggle conditional | Switch. When on, shows ConditionalRule inline (source question select + expected value). |
| Upload logo | file-dropzone, square crop hint. Preview header updates with uploaded logo. |
| Change accent color | Color picker or hex input. Preview updates all accent elements (stepper, CTA button). |
| Edit welcome text | Textarea. Preview header updates with new welcome text, debounced 300ms. |
| Edit slug | Input with validation (lowercase, hyphens only, 3-50 chars). URL preview updates. Uniqueness checked on blur via server action. |
| Save template | Primary button in header. Saves entire template state (questions + branding + slug) in single server action. Toast on success. Button disabled while saving. |
| Tab switch (Questions/Branding) | Client-side tab change. Preview always visible on desktop. |
| Mobile: toggle preview | On mobile, preview accessible via a "Voir l'apercu" button that opens a Sheet from bottom (full height). |

### Client-Facing Intake

| Interaction | Behavior |
|-------------|----------|
| Step navigation | "Suivant" validates current step fields with Zod before advancing. "Precedent" always available (no validation on back). |
| Stepper click | Click completed step to go back (only completed steps clickable). |
| Conditional field | Field appears/disappears with 200ms fade animation when condition met/unmet. |
| Submit | Final step "Soumettre ma demande" button. Full form validation. Response snapshot includes template schema. Toast: "Votre demande a ete envoyee." Redirect to confirmation page. |
| Field validation | Inline error messages below field, red border on field, aria-describedby linked. |
| Auto-save | localStorage auto-save every 10s for in-progress intake (keyed by slug + session). |

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Drag & drop | @dnd-kit provides keyboard support (Space to pick up, Arrow keys to move, Space to drop). Announce position via aria-live. |
| Question cards | role=listitem within role=list. aria-label includes question text and position ("Question 3 sur 7: Enfants"). |
| Color picker | Preset swatches are radio buttons with aria-label for color name. Hex input has label. Color contrast preview shown. |
| Template cards | role=radio within role=radiogroup. aria-checked on selected card. |
| Conditional fields | aria-expanded on conditional toggle. Hidden fields use display:none (removed from tab order). |
| Intake stepper | role=list with role=listitem per step. aria-current="step" on active step. Completed steps have aria-label "Etape terminee". |
| Form validation | Error messages linked via aria-describedby. aria-invalid="true" on errored fields. |
| Logo upload | aria-label="Telecharger le logo du cabinet". Alt text set to cabinet name after upload. |
| Preview panel | aria-label="Apercu du formulaire d'intake". role=region. |
| Color contrast | Lawyer accent color validated: if contrast ratio < 4.5:1 against white, show warning "Cette couleur peut etre difficile a lire. Choisissez une couleur plus foncee." |
